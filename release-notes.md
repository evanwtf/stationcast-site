# Release Notes

<p class="page-meta">What's new across recent TestFlight builds</p>

- [**12.1**](/release-notes/12.1/) — A big Siri and Spotlight upgrade on iPhone — name a station by voice, ask for the nearest one, a visual result card, Control Center control, and Spotlight search for your saved stations — plus StationCast arrives on the web
- [**10.23**](/release-notes/10.23/) — Map-first mode on iPhone, smarter map rain chips, dew point chart overlay, friendlier offline and rate-limited copy, plus the reimagined iPad layout with sidebar map and keyboard shortcuts, Alexa skill, expanded Siri intents, universal links, shareable QR codes, and a reorganized Settings screen
- [**10.12**](/release-notes/10.12/) — Map mode for picking stations on iOS, Android catching up to iOS, freeform station labels, auto-select your nearest saved station on launch, backend plausibility checks
- [**10.6**](/release-notes/10.6/) — Native Android app, Siri shortcut, inline metric sparklines, 15-minute temperature chart, friendlier distance formatting
- [**6.7**](/release-notes/6.7/) — Home screen widget, station distance, share-image polish, App Attest
